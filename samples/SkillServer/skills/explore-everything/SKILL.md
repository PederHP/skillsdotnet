# Explore Everything Server

This skill requires the **Everything Server** MCP server to be connected. When loaded,
the client host will automatically connect to it.

## What to do

1. Use the `echo` tool to validate connectivity to the Everything Server.
2. Tell the user what tools you have available in your tool list.

## Notes

- The Everything Server is an MCP reference server that exposes sample tools, resources,
  and prompts for testing purposes.
